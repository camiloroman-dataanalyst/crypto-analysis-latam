# 🪙 Crypto Analysis — Mercado en Tiempo Real

## 📌 Descripción
Análisis del mercado de criptomonedas en tiempo real usando 
la API de CoinGecko. Extracción, limpieza y visualización 
de datos con Python y dashboard interactivo en Power BI.
Proyecto desarrollado con datos reales extraídos el 29 de 
abril de 2026.

## 🎯 Objetivo
Analizar el comportamiento del mercado crypto — identificando 
tendencias, ganadores, perdedores y el comportamiento histórico 
de Bitcoin y Ethereum en los últimos 30 días.

## 📊 Hallazgos principales
- **Solo el 20% del mercado está en verde** — 10 de 50 criptos 
  subiendo el día del análisis
- **Bitcoin subió 13.36%** en los últimos 30 días
- **Ethereum subió 10.49%** en los últimos 30 días
- **Precio máximo de BTC:** $78,645 USD el 26 de abril
- **Precio mínimo de BTC:** $66,699 USD el 31 de marzo
- **Dogecoin** fue la única cripto del top 10 en verde ese día
- **Bitcoin domina** el mercado con una capitalización de 
  $1.5 billones USD

## 🛠️ Herramientas utilizadas
- **Python** — extracción y análisis de datos
- **Pandas** — limpieza y transformación
- **Matplotlib / Seaborn** — visualizaciones estáticas
- **Jupyter Notebook** — documentación del análisis
- **Power BI** — dashboard interactivo
- **CoinGecko API** — datos en tiempo real
- **GitHub** — control de versiones

## 🔗 Fuente de datos
CoinGecko API — datos de mercado en tiempo real  
[coingecko.com](https://www.coingecko.com)

## ⚠️ Limitaciones
- Los datos son una fotografía del mercado en un momento 
  específico — el análisis no es predictivo
- La API gratuita de CoinGecko tiene límite de llamadas 
  por minuto
- Los precios cambian constantemente — los hallazgos 
  reflejan el estado del mercado el 29 de abril de 2026
